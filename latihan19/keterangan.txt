---- Keterangan Latihan 19 (MongoDB Contact App dengan Mongoose (Bagian 1)) ----

1. Pengertian
   - Mongoose
     - Digunakan untuk mempermudah dari aplikasi express yang dibuat terhubung ke MongoDB
     - Disebut dengan object modeling, atau istilahnya object document model atau ODM

2. Cara Penggunaan Installasi
   - Ketik npm install express@4.17.1
     - Untuk menginstall express js sesuai dengan versi pada dependensi
   - Ketik npm install ejs@3.1.6
     - Untuk menginstall ejs sesuatu dengan versi pada dependensi
   - Ketik npm install express-ejs-layouts@2.5.0
     - Untuk menginstall express ejs layouts sesuai dengan versi pada dependensi
   - Ketik npm install mongoose@5.12.13
     - Untuk menginstall mongoose sesuai dengan versi pada dependensi
   - Ketik npm install espress-session@1.17.2
     - Untuk menginstall express session sesuai dengan versi pada dependensi
   - Ketik npm install cookie-parser@1.4.5
     - Untuk menginstall cookie parser sesuai dengan versi pada dependensi
   - Ketik npm install connect-flash@0.1.1
     - Untuk menginstall connect flash sesuai dengan versi pada dependensi
